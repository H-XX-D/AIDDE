# DD and DeeDee

DD, also called DeeDee, is the companion identity for AIDDE.

## Role

DD is the visible surface of the environment:

- title-bar face,
- app icon,
- dock icon,
- sidecar handle,
- voice target,
- status indicator,
- reward avatar,
- future 3D printable collectible.

## State language

DD should communicate state simply:

- awake,
- asleep,
- listening,
- thinking,
- blocked,
- idle,
- pinned,
- sidecar-hidden.

The face should be useful first and charming second. Blinks and expression
changes are valuable because they make state feel alive, but they should never
confuse critical controls.

## Collectible surface

DD can wear accessories and cosmetics unlocked by:

- daily use,
- shipping milestones,
- learning missions,
- tool mastery,
- hidden actions,
- seasonal events,
- developer-only milestones.

The long-term idea is that DD accessories can become printable physical objects.
Users should eventually be able to export selected DD accessories as STL files
for 3D printing.

## Brand note

DD should stay visually simple and distinctive:

- rounded form,
- two dark eyes,
- small smile,
- accessory-friendly silhouette,
- readable at tiny sizes.
